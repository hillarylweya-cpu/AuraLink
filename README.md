# AuraLink - Secure Communication Platform

## 🚀 Deploy to Streamlit Cloud

1. Go to https://share.streamlit.io
2. Click "New App"
3. Select: `hillarylweya-cpu/AuraLink`
4. Branch: `main`
5. File: `streamlit_app.py`
6. Click Deploy

**Done!** Your app will be live in 2-3 minutes.

## 📱 Features

- ✅ Secure messaging with chat history
- ✅ File transfer
- ✅ Voice call UI
- ✅ End-to-end encryption
- ✅ Export chat as JSON
- ✅ Mobile responsive
- ✅ No errors

## 💻 Run Locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

Open: http://localhost:8501
